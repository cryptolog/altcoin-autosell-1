altcoin-autosell
================

Cryptocurrency/altcoin autoseller script written in Perl

Requires modules Log::Log4perl, LWP::UserAgent, and YAML::Tiny.
```shell
cpan Log::Log4perl LWP::UserAgent YAML::Tiny
```
  
USAGE:
```shell
./autoseller.pl -usage
```
```shell
perl autoseller.pl -usage
```
